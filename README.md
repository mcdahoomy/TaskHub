# TaskHub

A clean, lightweight task management web app built with **vanilla HTML, CSS, and JavaScript**.

This project focuses on **real-world functionality**, clean structure, and maintainable code — without using any frameworks or external libraries.

---

## ✨ Features

- Create, edit, and delete tasks
- Optional notes (editable)
- Priority levels: Low / Medium / High
- Due dates with overdue highlighting
- Search tasks by title or notes
- Filters: All / Active / Completed
- Persistent storage using `localStorage`
- Export tasks as JSON
- Import tasks from JSON backup
- Keyboard-friendly editing (Enter / Escape)
